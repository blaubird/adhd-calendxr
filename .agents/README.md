# CALENDXR Repo Skills

This directory contains repo-scoped Codex skills for recurring CALENDXR work. Future prompts can be shorter by referencing the relevant skill:

- `Use $calendxr-telegram` for Telegram bot commands, settings, language, digest, reminders, formatting, and callbacks.
- `Use $calendxr-calendar` for Calendar item logic, move/reschedule, recurrence, occurrence/series behavior, and daily/weekly item queries.
- `Use $calendxr-db-migrations` for Drizzle, Neon Postgres, migrations, env vars, and production schema safety.
- `Use $calendxr-ui` for web UI polish and the dark glass command-center direction.
- `Use $calendxr-task-workflow` for phased task prompts, guardrails, validation checklists, and deliverables.
- `Use $calendxr-openai-usage` for OpenAI API, model, prompt, AI draft parsing, and intent-routing work.
